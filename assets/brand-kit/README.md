# Brand kit — assets génératifs prêts à l'emploi

Ressources **générées** spécifiquement pour faciliter la création de slides Pluxee. Différent du reste de `/assets/` qui contient des fichiers crawlés depuis pluxee.fr.

## Contenu

```
brand-kit/
├── x-mark/              12 SVG (6 solid + 6 keyline) dans la palette officielle
└── holding-shapes/      30 SVG (incision + chamfer dans 5 couleurs)
```

## Couleurs officielles utilisées

| Nom | HEX | Usage |
|---|---|---|
| Deep Blue | `#221C46` | Texte primaire & fond signature |
| Boldly Blue | `#17CCF9` | Secondaire vibrant |
| Ultra Green | `#00EB5E` | Signature — doser ≤ 10 % surface |
| Very Yellow | `#FFDC37` | Énergie / highlights |
| Confidently Coral | `#FF7375` | Punch émotionnel |
| White | `#FFFFFF` | X-mark white = uniquement cards & vouchers |
