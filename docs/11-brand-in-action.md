# 11 — Brand in action

> Source : Brand book v1.4, pages 151–167.

Cette section synthétise comment la marque s'applique sur **tous les supports** : POS, signage, digital, motion, templates.

## Merchant stickers (p. 152)

Stickers acceptance "Pluxee acceptés ici" pour vitrines de commerçants.
- **Forme circulaire ou rectangulaire** avec X-mark.
- **Couleurs vives** (Ultra Green, Boldly Blue, Very Yellow, Coral) pour maximum de visibilité en vitrine.
- Texte : "Get that Pluxee feeling", "Live joyful", "Enjoy!" selon variante.
- Voir page 48 du brand book pour exemples (Message amplifier).

## Construction (p. 153)

Règles de construction des supports brand :
- Grille basée sur la hauteur du x du logo Pluxee.
- Marges cohérentes : 2× la hauteur du x sur les bords.
- Logo toujours dans un coin (top-left ou top-right le plus souvent).
- X-mark comme élément graphique central ou ponctuation.

## Brand amplifier (p. 155)

Le concept d'**amplifier** = utiliser le X-mark comme **mégaphone graphique** d'un message court.
- Cf. [03-xmark.md](03-xmark.md#02-message-amplifier--règles).
- Idéal pour OOH (affichage), posters, splash screens, motion intros.

## Brand examples (p. 160)

Le brand book montre des exemples d'application complète :
- **OOH / billboards** : full color background + X-mark central + 1 statement
- **Web banners** : photo + X-mark connector + tagline
- **App screens** : Deep Blue UI + accents Ultra Green sur CTAs
- **Email** : header logo + photo + body Deep Blue
- **Signage** : "Pluxee accepted here" en vitrine
- **Goodies** : tote bags, mugs, stickers, t-shirts brandés

## Applying the brand sustainably (p. 161)

Mention RSE :
- Print : encres végétales, papier FSC/recyclé.
- Goodies : matériaux durables (coton bio, recyclé).
- Limiter les supports physiques quand un digital est possible.
- Charte cohérente avec la stratégie CSR Pluxee.

## Templates (p. 164)

Le brand book fait référence à des **templates téléchargeables** :
- PowerPoint
- Word
- Email (HTML + signature)
- Réseaux sociaux (Instagram, LinkedIn, Facebook, X)

> ⚠️ Templates accessibles sur le **brand portal interne Pluxee** — il faudrait demander un accès au brand team pour récupérer le template PPT officiel (ce qui éviterait de le reconstruire from scratch).

## Motion principles (p. 165)

Pour les vidéos / GIFs / animations :
- **Easing** : `cubic-bezier` doux (in-out), pas de bounce excessif.
- **Durations** : 300–600 ms pour micro-interactions, 1.5–3 s pour transitions hero.
- **Le X-mark s'anime** souvent en :
  - *Stamp in* (apparaît au centre avec scale 0→1)
  - *Rotate* léger (jamais 360°)
  - *Pulse* (scale 1 → 1.05 → 1)
- **Texte typing/reveal** : par mots, pas par lettres.
- **Couleurs** : pas de gradient animé. Transitions de couleur OK si entre couleurs palette.
